Why Navigation in SwiftUI is Painful

❌ String-based routing leads to runtime crashes
❌ Hard to scale for large apps
❌ Deep linking becomes messy
❌ Poor separation of concerns

The NavigationPilot Approach

🔒 Fully type-safe navigation
🧠 Clean architecture friendly
🔗 Built-in deep linking
⚡ Minimal boilerplate

Simple API

navigationPilot.push(.detail(id: 1))

How It Works

NavigationPilot introduces a strongly-typed routing system using enums. Each route represents a destination in your app.

This ensures compile-time safety and eliminates invalid navigation states.

enum Route { case home case detail(id: Int) }
